# Agent Planet 页面卡片信息

## 一、引擎管理（原服务管理）

文件：`src/pages/services.js`

### 1. 版本卡片
- 显示当前 OpenClaw 版本号
- 显示版本来源标签（汉化优化版 / 官方原版）
- 版本状态提示：已对齐推荐版 / 高于推荐版 / 偏离推荐版 / 无推荐版
- 最新上游版本提示
- 操作按钮：回退到推荐版 / 切换到推荐版、切换到汉化版/官方版
- 版本策略说明文字
- Docker 部署模式：显示不同布局，含 docker pull 命令

### 2. Gateway 服务卡片
- 服务名称：`ai.openclaw.gateway`
- 运行状态指示灯（运行中 / 已停止 / 警告）
- 进程 PID 显示
- CLI 未安装提示 + 安装命令
- 外部 Gateway 检测警告 + 认领/查看引导按钮
- 操作按钮：启动 / 停止 / 重启 / 安装 / 卸载（macOS 专属）
- 轮询等待状态（带计时、取消按钮）

### 3. Docker 管理
- Docker 概览状态栏（在线节点数、容器数）
- 操作按钮：刷新 Docker、添加节点、拉取镜像、创建容器
- 节点卡片列表（节点名、端点、Docker 版本、内存、CPU）
- 每个节点下的容器列表（容器名、镜像、状态、端口）
- 容器操作：启动 / 停止 / 重启 / 删除
- 默认镜像提示

### 4. 配置文件编辑器
- JSON 语法高亮的 textarea 编辑器
- 实时 JSON 语法检测
- 未保存修改状态提示
- 操作按钮：保存并重启 / 仅保存 / 重新加载

### 5. 配置校准
- 继承校准按钮 + 说明
- 完全初始化修复按钮 + 说明
- 校准状态显示

### 6. 配置备份管理
- 创建备份按钮
- 备份列表（备份名、日期、大小）
- 操作：恢复 / 删除

- QQ 群二维码 + 微信群二维码 + 抖音群二维码 + 飞书群二维码
- 欢迎语（中英文）
- 加入按钮：Discord / QQ / 微信 / 抖音 / 飞书 / 元宝

#### 3. 相关项目
- OpenClaw — GitHub
- OpenClaw-zh — GitHub
- Agent Planet — GitHub + Gitee 国内镜像
- ClawApp — GitHub
- cftunnel — GitHub

#### 4. 参与贡献

#### 7. 底部信息
- 技术栈说明
- 版权信息

### Hermes 引擎模式

#### 1. 版本信息卡片（3 张）
- **Agent Planet 版本卡片**：同上

- **Python 卡片**：Python 版本 + 路径

> Hermes 模式下不展示社群、项目、贡献、链接、关于我们区块

### ZeroClaw（新天）引擎模式

#### 1. 版本信息卡片（3 张）
- **Agent Planet 版本卡片**：同上
- **ZeroClaw 卡片**：Windows 平台标识 + 下载 / 访问官网按钮
- **相关链接卡片**：xtclaw.xtnet.cc + 使用帮助链接

> ZeroClaw 模式下不展示社群区块
